Sample application and modules for backend and frontend usage of typeorm

**Installation**
Create link on backend

```
cd backend
npm link
npm install
tsc
```

Create link on frontend

```
cd frontend
npm link
npm install
tsc
```

Use links from backend and frontend in the example Application

```
cd example-app
npm link backend
npm link frontend
npm install
```

**Run Application**
```
cd example-app
npm start
```
